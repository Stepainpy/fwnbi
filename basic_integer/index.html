template <
    size_t Bits,
    class  DigitT,
    bool   Signed
> class basic_integer;
Template parameters
Bits count of bits in integer, Bits must be a multiple of bit width of DigitT
DigitT type, used for storage digits of integer
Signed signedness, if equal to true, then integer is signed, else integer is unsigned

The class template emulating semantic of integer with signedness depended on Signed and bit width equal to Bits. Digits order is little i.e. first digit with index 0 is least signification digit. For signedness used two's complement method.
