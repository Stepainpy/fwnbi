Type alias uN (where N is 8, 16, 32 and 64) is uint_fastN_t if UINT_FASTN_MAX equal to UINTN_MAX, otherwise is uintN_t.

This aliases using as template parameter DigitT in basic_integer. If available GCC extension with type __uint128_t you can use u64 as valid value in DigitT, otherwise only u8, u16 and u32.
